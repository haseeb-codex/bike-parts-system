# Bike Spare Parts Business Management System

A comprehensive web application for managing bike spare parts manufacturing business operations.

## Project Structure

- `/docs` - Documentation files
- `/backend` - Node.js/Express API server
- `/frontend` - React web application

## Getting Started

### Prerequisites
- Node.js v16+ (LTS)
- MongoDB (local instance running)
- npm or yarn

### Installation

1. Backend Setup:
   cd backend
   npm install
   npm start

2. Frontend Setup:
   cd frontend
   npm install
   npm start
